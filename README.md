# 🛳️ Titanic EDA (Exploratory Data Analysis)

## 📌 Project Overview

In this repository, I am performing EDA on the famous Ttianic dataset to uncover patterns, trends, and key factors that
influenced passenger survival.

The goal is to use data analysis and visualization techniques to answer questions like:-

- Who was more likely to survive?
- Did class, gender, or age play a role?
- What patterns can we observe from the data?

## 📂 Dataset

The dataset I used in this project contains information about Titanic's passengers, including:
- Passenger details (age, gender, class)
- Ticket and Fare information
- Cabin and Embarkation details
- Survival status
